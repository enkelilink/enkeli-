Enkelilink

Guardian Angel Network for Crisis Communication

The Mission

Enkelilink is a free, open-source, privacy-first resilient communication network. We are building a tool that allows smartphones to communicate directly during disasters, in conflict zones, and in remote areas without any cellular data or internet connection.

How It Works

Peer-to-Peer Mesh

Phones connect directly using Bluetooth & Wi-Fi, creating a network that grows with every user.

Hybrid Infrastructure

Mobile and solar-powered units act as super-nodes, extending the network's range and lifespan.

Emergency Features

Priority messaging, resource mapping, and a secure system for emergency responders.

Radio Integration

Broadcasts critical alerts via AM/FM, reaching everyone, even without a smartphone.
